## OPACHelper

:sunny:本工具旨在通过命令行抓取OPAC图书馆书籍借阅信息...╮(╯▽╰)╭)

## 如何使用
1.确保计算机安装有tesseract  
2.执行`npm install`或`yarn`安装项目所需依赖  
3.在`config.json`填入图书馆的访问地址、学号和密码  
4.执行`npm start`或`yarn start`来获取图书信息吧~

## 本小助手有以下特性呢

* 通过识别验证码直接登陆图书馆
* 抓取当前个人的图书书籍借阅信息
* Cookie过期后自动获取新Cookie
